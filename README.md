# jsonkdl

Converts JSON to KDL.

## Usage

```
JSON to KDL Converter
Usage:
  jsonkdl <input> <output> [options]
Arguments:
  <input>   Input JSON file
  <output>  Output KDL file
Options:
  -f, --force      Overwrite existing files
  -v, --verbose    Verbose output
  -h, --help       Show this help message
```
